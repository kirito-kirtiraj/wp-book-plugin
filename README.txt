rtCamp's WordPress plugin development assignment.

== Description ==

The plugin does the following:
* Registers custom post type for books.
* Registers book category.
* Registers book tag.
* Creates a custom meta box to input book information.
* Creates a custom meta table to save book information.
* Creates some custom admin settings under book post type.
* Creates a book shortcode.

Used "WORDPRESS PLUGIN BOILERPLATE GENERATOR" from https://wppb.me/ to create the basic skeleton of plugin.
Used phpcs to follow best practices as much as possible.
